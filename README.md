# Automata öntözőrendszer ESP32 segítségével

Bemutató videó linkje: https://drive.google.com/file/d/1XHb1RlbuTiZXZn4gKcPo2RB6jqGnzEKh/view

## Projekt célja

A projekt célja egy IoT alapú automata öntözőrendszer elkészítése volt ESP32 mikrokontroller segítségével.

## Funkciók

- Talajnedvesség mérés
- Automatikus öntözés
- Vízszint figyelés
- Pumpa vezérlés MOSFET segítségével
- Blynk alapú távfelügyelet
- Hibakezelés

## Felhasznált alkatrészek

- ESP32
- Kapacitív talajnedvesség szenzor
- Úszókapcsolós vízszint érzékelő
- MOSFET modul
- 5V vízpumpa
- Blynk
